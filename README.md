# Proyecto: Cifrados Clásico

## Descripción
Sitio web para cifrar y descifrar texto con **4 algoritmos clásicos**:
- César  
- Vigenère  
- Transposición Columnar  
- Atbash

# Tecnologías
HTML, CSS, JavaScript (Frontend puro).

## Cómo ejecutar
1. Clona este repositorio o descarga los archivos.
2. Abre `https://cifrado-e8b34.web.app/` en tu navegador.
3. Ingresa un texto, clave (si aplica), elige el método y presiona *Cifrar* o *Descifrar*.
